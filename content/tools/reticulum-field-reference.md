## Features

**Verified Content** - All public nodes, commands, and configurations tested on LFM production infrastructure.

**Works Offline** - After first load, the entire guide is cached. Access it anywhere without internet.

**3 Calculators** - LoRa link budget, airtime calculator, and interface config generator.

## Sections

1. **Reticulum Core** - Why Reticulum, quick start, architecture, ecosystem versions
2. **Network Interfaces** - TCP, I2P, RNode, KISS, AX.25, Serial, Pipe configurations
3. **Hardware** - Supported RNode boards, HF radio setup (Xiegu G90 + FreeDV), flashing, regulations
4. **NomadNet** - Node hosting, Micron markup, static/dynamic pages, Matrix bridge
5. **LXMF** - Messaging protocol, delivery methods, clients, LXMFMonero verified transactions
6. **Utilities** - rnstatus, rnpath, rnsd, rncp, rnodeconf, rnid command reference
7. **Troubleshooting** - Quick fixes, common problems, debug mode
