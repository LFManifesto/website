## Features

**Full-Text Search** - Find anything across all content instantly. Press Ctrl+K for quick access.

**Interactive Checklists** - Pre-flight OPSEC, in-flight protocols, landing procedures, spotter duties, and more. Progress saves automatically.

**Payload Delivery Request Generator** - Build formatted 5-Line requests for sUAS attack, smoke, or resupply missions.

**Works Offline** - After first load, the entire guide is cached. Access it anywhere without internet.

## Sections

1. **Drones** - UAS protocols, operational considerations, TTPs, payload delivery, CUAS techniques, FPV tactics
2. **Communications** - DMR radio operations, G90 & JS8Call setup
3. **SIGINT** - Software Defined Radio fundamentals, RF emitter analysis
4. **Marksmanship** - Target detection, range estimation, wind reading, rapid engagement, team communication

## Quick Reference Tables

The guide includes formatted tables for:

- UAS frequency bands and protocols
- Camouflage patterns by range and conditions
- Wind value by clock position
- Mil-relation target sizes
- SDR hardware comparison
- Shot correction calls
